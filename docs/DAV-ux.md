# DAV UX Notes

This project applies SESMag-oriented design choices across every page, with special attention to DAV-style needs:

- visible labels for every field
- plain-language status text such as "Waiting for manager review"
- large touch targets
- mobile-first page layouts
- visible focus states
- icons plus text for status, not color alone
- public versus private profile fields explained clearly
- low-friction approval and edit flows

The goal is to reduce ambiguity, cognitive load, and navigation stress while keeping the dashboard and workflows usable for different comfort levels with technology.
