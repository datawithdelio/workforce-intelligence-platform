# Career Assets

## Resume bullets

- Built a full-stack workforce analytics platform with `Next.js 14`, `Express`, `PostgreSQL`, and `Drizzle`, implementing role-based employee self-service, manager approval workflows, KPI dashboards, notifications, and audit logging across a monorepo architecture.
- Designed an analytics and ML layer using SQL reporting views, automated KPI snapshot jobs, reminder workflows, and a Python scikit-learn completion-risk model that writes explainable workforce risk scores back into PostgreSQL.

## LinkedIn / portfolio project description

Workforce Intelligence Platform is a full-stack PeopleOps application I built to combine employee self-service, manager approvals, analytics dashboards, automation workflows, and applied machine learning in one product. The stack uses Next.js, Express, PostgreSQL, Drizzle, and Python, and the UX was designed with accessibility and lower-friction interaction in mind for a wide range of users.

## Short interview pitch

I built a workforce management platform that sits at the intersection of full-stack engineering, analytics engineering, and applied data science. Employees submit profile updates, managers approve changes through an auditable workflow, admins monitor workforce KPIs, and a Python scoring pipeline writes explainable completion-risk signals back into the app. It gave me a chance to work across product design, backend APIs, relational data modeling, reporting, automation, and ML integration in one cohesive system.
